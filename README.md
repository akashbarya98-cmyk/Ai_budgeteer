# 📌 Overview

AI Budgeteer is a modern web application that helps organizations and teams manage their AI-related expenses efficiently.

As AI adoption grows, companies often struggle with:

- Uncontrolled AI spending
- Excessive token consumption
- Lack of visibility into usage patterns
- Inefficient model selection
- Budget overruns

AI Budgeteer provides real-time analytics, budget monitoring, prompt optimization, and efficiency tracking to maximize value while minimizing costs.

Monitor your AI budget in real time with spending analytics, burn rate tracking, budget utilization, and alerts.

- Monthly budget monitoring

- Cost trend visualization

- Team rankings

Configure budget thresholds and automated alerts.

- Monthly budget limits
- Multi-level alert thresholds
- Critical budget warnings
- Cost-control mechanisms

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/akashbarya98-cmyk/Ai-Budgeteer.git
cd Ai-Budgeteer
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

# 🔨 Build Production Version

```bash
npm run build
npm run start
```
